# VendingMachine
A UI-less vending machine using jest for TDD.

## Installation
Download the repo, then run:

```sh
$ npm install
$ npm run test
```

### Features
Features of this vending machine include:
- Printing Inventory
- Restocking Inventory
- Re-supplying Chnage
- Dispensing items
- Returning change as coins
